Una herramienta de interfaz web simple y eficiente desarrollada en JavaScript para la manipulación de códigos QR. Este proyecto permite a los usuarios transformar rápidamente enlaces, textos o datos de contacto en imágenes QR escaneables en cuestión de segundos.

---

## ✨ Características Principales

* 🔄 **Generación en Tiempo Real:** Crea códigos QR instantáneamente a medida que el usuario escribe el texto o pega la URL.
* 📥 **Descarga Fácil:** Funcionalidad para exportar y descargar el código QR generado en formato de imagen (PNG/JPG) con un solo clic.
* 🎨 **Personalización (Opcional):** [Añade esto si tu código lo permite: Cambio de colores, tamaño del código y niveles de corrección de errores].
* 📸 **Escáner Integrado (Opcional):** [Añade esto si aplica: Capacidad de leer códigos QR directamente usando la cámara web del dispositivo].
* 📱 **Diseño Responsivo:** Interfaz adaptable que funciona perfectamente tanto en dispositivos móviles como en computadoras de escritorio.
